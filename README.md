# Prodigy Infotech: Machine Learning - Task 01

## 🏡 House Price Prediction using Linear Regression

This repository contains the implementation of Task 01 for the Prodigy Infotech Machine Learning Internship. The goal of this project is to build a linear regression model that predicts house prices based on fundamental features: square footage, number of bedrooms, and number of bathrooms.

### 📋 Project Overview
Predicting real estate prices accurately is a classic machine learning problem. This project demonstrates how to implement a multiple linear regression model to find the relationship between a house's physical attributes and its final sale price. 

To make this code easily reproducible, the script generates a realistic synthetic dataset internally. This allows anyone to clone the repository and run the code immediately without downloading external files.

### 🛠️ Technologies & Libraries Used
* **Python 3.x:** Core programming language.
* **Pandas & NumPy:** For data generation, manipulation, and structuring.
* **Scikit-Learn:** For building, training, and evaluating the Linear Regression model.
* **Matplotlib & Seaborn:** For visualizing the actual vs. predicted prices.
